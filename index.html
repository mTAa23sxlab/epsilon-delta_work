ε-δ論法の定義
limx→a f(x) = L ⟺ ∀ε > 0, ∃δ > 0 s.t. |x − a| < δ ⇒ |f(x) − L| < ε

意味:

  • どんなに小さい正の数 ε をとっても
  • それに対応する正の数 δ が必ず見つかって
  • x が a から δ 未満の距離にあれば、f(x) は L から ε 未満の距離にある
💡 ポイント: ε が先に与えられ、それに応じて δ を選ぶ
例題: limx→1 4x² = 4 の証明
証明

どんな ε > 0 に対しても，

0 < |x − 1| < δ ⟹ |4x² − 4| < ε

が成り立つような δ が存在することを示せばよい。

ε > 0 をどんなに小さくとっても、それに合う δ を選べる。

|4x² − 4| = 4|x² − 1| = 4|x − 1||x + 1|

|x − 1| < δ < 1 と仮定すると、|x + 1| < 3 なので

|4x² − 4| < 4 · δ · 3 = 12δ

12δ < ε となるように δ = min(1, ε/12) と選べばよい。

連続の定義

ε-δ論法で書くと:

∀ε > 0, ∃δ > 0 s.t. |x − a| < δ ⇒ |f(x) − f(a)| < ε
f(x) が点 a で連続 ⟺ limx→a f(x) = f(a)
💡 「どんなに小さい ε をとっても δ が必ず見つかる」= 連続
💡 「あるεをとったとき δ が見つからない」= 不連続
